
# ☕ Estudos de  Java

Este repositório é o meu "laboratório" de estudos em Java. Aqui eu documento todos os exercicios feitos em Java, desde a lógica fundamental até estruturas um pouco mais complexas.


## 🖥️ Objetivos do Projeto

- Praticar lógica de programação e algoritmos de forma diária.

- Documentar todo o meu progresso de aprendizado em Java

---

## 📂 Estrutura do repositório


O projeto está organizado por módulos para facilitar a navegação e o estudo progressivo:

- **01 - Lógica e Fundamentos:** Exercícios iniciais focados em sintaxe, operadores, condicionais e laços de repetição (ex: Cálculo de área, Bhaskara, Loja Online).

- **02 - Estrutura de Dados:** Implementações que exigem organização de dados, como arrays (vetores), matrizes e manipulação de listas.

## 🚀 Como executar os exercícios

Para rodar qualquer um dos exercícios localmente, você precisa ter o JDK instalado. No terminal, use:

```python

# Compilar o arquivo
javac NomeDoArquivo.java

# Executar
java NomeDoArquivo

```
